These modules are required to use the ASSP_Razor Plugin.

To install the modules:

on windows

>ppm install razor-agents.ppd


on MAC and nix:

- unzip and untar  razor-agents.src.tar.gz
- cd in to the directory razor-agents-2.84_assp and run the following command sequence
>perl Makefile.PL
>make
>make test
>make install
